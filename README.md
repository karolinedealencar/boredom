# Boredom

Generate a ramdom activity to cure boredom.

## Getting Started

### Installing

```
git clone git@github.com:karolinedealencar/boredom.git
cd boredom
npm install
```

## How to run

```
npm start
```

## Built With

- [React](https://reactjs.org/)
- [Bored API](https://www.boredapi.com/)
